Midterm Instructions:

Add semantic tags to your index page: header, nav, main, footer.

Create the grid layout shown above. Pick background colors of your choice.

Note that there are 5 sections in the grid, but 4 semantic tags!!! How are you going to handle this? One way is to have two header tags with different class names (one for the logo and one for text)

You only need ONE div for your container. Make sure that your grid takes up the entire screen. Use percentages and not pixels.

No inline or internal styles. All CSS must be in the included CSS file.

Add image, header text, nav and footer info.

Please recall that all text still has to be marked up with h1, h2, p, etc. Last date modified in footer using Javascript.
